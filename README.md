# 100Hires – Initial Setup Assignment

## 📌 Overview
This repository documents the completion of the initial setup task assigned as part of the 100Hires hiring process. The objective of this task was to install required development tools, configure the environment, and demonstrate the ability to follow instructions, troubleshoot issues, and document the process clearly.

---

## 🛠 Tools Installed

- Cursor IDE
- Claude Code extension (Anthropic)
- Codex extension (OpenAI)
- GitHub (for repository hosting and version control)

---

## ⚙️ Steps Completed

1. Installed Cursor IDE from the official website.
2. Launched Cursor and completed account setup.
3. Connected GitHub account to Cursor for repository access.
4. Installed the Claude Code extension from the extensions marketplace.
5. Installed the Codex extension from the extensions marketplace.
6. Created a new public GitHub repository named **Portfolio** to document the assignment.
7. Cloned and opened the repository in Cursor IDE.
8. Created and edited the README.md file to document the process.
9. Committed the changes locally.
10. Pushed the repository to GitHub.

---

## ⚠️ Challenges Faced

- Initial confusion during GitHub authorization process.
- Redirection to browser during login created uncertainty about workflow.
- Difficulty identifying the correct extensions among multiple similarly named options.
- Understanding how to commit and push changes from Cursor IDE.

---

## ✅ Solutions Implemented

- Followed official documentation and on-screen instructions carefully.
- Verified extension publishers (Anthropic for Claude Code, OpenAI for Codex) before installation.
- Completed authentication via browser and returned to Cursor to continue workflow.
- Re-attempted steps patiently and ensured correct sequence of actions.
- Used online resources and tutorials to clarify GitHub and version control steps.

---

## 🎯 Outcome

Successfully completed the setup process within the given timeframe and requirements.  
This repository serves as proof of:

- Ability to follow technical instructions
- Problem-solving and troubleshooting skills
- Familiarity with modern development tools and workflows
- Clear and structured documentation practices
